# Frontend
- Astro 5
  Opis: Pozwala na tworzenie szybkich, wydajnych stron i aplikacji z minimalną ilością JavaScript.
- React 19
  Opis: Zapewnia interaktywność i komponenty klienckie tam, gdzie potrzeba dynamicznych UI/ów.
- TypeScript 5
  Opis: Zapewnia statyczne typowanie kodu, lepsze wsparcie IDE.
- Tailwind 4
  Opis: Narzędzie do stylowania z utility-first podejściem, pozwalające szybko budować responsywne i spójne interfejsy.
- Shadcn/ui
  Opis: Zestaw gotowych komponentów React (design system), który przyspieszy budowę spójnego i dostępnego UI.
# Backend
- Supabase
  Opis: Kompleksowe rozwiązanie backendowe oparte na PostgreSQL. Zapewnia bazę danych, SDK, uwierzytelnianie użytkowników oraz funkcje BaaS.
# AI
- Openrouter.ai (integracja)
  Opis: Warstwa pośrednicząca do komunikacji z wieloma modelami (OpenAI, Anthropic, Google itd.), umożliwiająca elastyczny wybór modeli oraz zarządzanie limitami i kosztami API.
# CI/CD i Hosting
- GitHub Actions
  Opis: System CI/CD do automatyzacji buildów, testów i wdrożeń.
- DigitalOcean (Docker)
  Opis: Platforma hostingowa, na której aplikacja może być wdrożona jako kontener Docker; proste i skalowalne podejście dla MVP.
